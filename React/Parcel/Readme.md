First of All you have to install npm in the directory
command is: 
npm init -y

Then you have to install Dependencies like React & reactdom
commands are
npm install react
npm install react-dom


then you have to install bundler say vite or parcel or any other
it will add parcel in your dependencies

commands are
npm install parcel

then build your project

to run that particular project
open terminal and type the following command

npx parcel index.html

it will give you localhost link


if you want to get a production level code
then type the following command

npm build parcel index.html
